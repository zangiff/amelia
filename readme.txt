Объемный, полностью адаптивный HTML-CSS-JS-проект интернет-магазина на 15 страниц, с логикой и ограниченным рамками JS функционалом, реализованный с использованием Gulp 4 и препроцесоора SASS. Bootstrap 4 присутствует в ограниченных масштабах, без сетки и подавляющего большинства других функций. Тут есть все самое насущное - иконки FontAwesome, подключаемые как через тег <i>, так и через псевдоэлементы, модальные окна, страницы-калькуляторы, выпадающие списки, bootstrap dropdown, стилизация селектов, табы, разнообразно реализованные формы регистрации, масса hover-эффектов на JS и SASS.Все это с достаточно обширным Javascript-функционалом: код файлов некоторых JS-скриптов, подключенных только к одной из страниц, превосходит 100 строк. Весь JS-код предлагается на нативном JS. JQuery и другие библиотеки в этом проекте не использовались.
